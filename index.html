Attendance Checker

Project Abstract

โครงการ Attendance Checker มีจุดมุ่งหมายเพื่อแก้ไขปัญหาในการเช็คชื่อของนักศึกษาที่ไม่สามารถดำเนินการสแกนหรือเช็คชื่อได้ทันเวลาในชั้นเรียน รวมทั้งอาจนำไปใช้เพื่ออำนวยความสะดวกในการเช็คชื่อก่อนเข้าห้องสอบได้อีกด้วย ระบบนี้ออกแบบมาเพื่อช่วยนักศึกษาที่เผชิญปัญหาการสแกน QR Code ไม่ทันหรือเกิดข้อผิดพลาดที่ทำให้ไม่สามารถเช็คชื่อได้สำเร็จ โดยมีการแจ้งเตือนผลการเช็คชื่อผ่านแอปพลิเคชัน Line เพื่อให้ผู้ใช้งานรับทราบ การทำงานของโปรเจกต์เริ่มต้นจากการสแกนบัตรของนักศึกษากับเครื่องตรวจบัตร เมื่อมีการสแกนบัตร ระบบจะตรวจสอบและบันทึกข้อมูลการเช็คชื่อไว้ พร้อมทั้งส่งการแจ้งเตือนไปยัง Line โดยระบุเวลาที่ทำการเช็คชื่อ ต่อจากนั้น ข้อมูลบัตรที่สแกนได้จะถูกส่งไปเปรียบเทียบกับฐานข้อมูลบน Google Sheet เพื่อยืนยันตัวตนและแสดงข้อมูลชื่อ นามสกุล รหัสนักศึกษา สถานะ วันที่และเวลาที่ทำการเช็คชื่อ หากบัตรที่สแกนไม่สามารถระบุตัวตนได้ ระบบจะขึ้นข้อความว่า "Unknown" เพื่อแจ้งเตือน

Key Features:

  • มีการเชื่อมต่อกับ Line เพื่อแสดงให้ผู้ใช้งานรับทราบ
  • เก็บข้อมูลการสแกนบัตร ไว้ใน Google Sheet

การทำงานของ Attendance Checker

1

Card Scanning

นักศึกษาสแกนบัตรเข้ากับเครื่องสแกนบัตร:

  • RFID card reader
  • RFID Tag (Student ID card)
  • Real-time scanning
RFID Reader Arduino UNO R4 WIFI
2

Data Verification

เครื่องสแกนทำการตรวจสอบว่ามีบัตรมาสแกนหรือไม่ถ้ามีส่งไปแจ้งเตือนที่ Line พร้อมทั้งแสดงข้อมูลการเช็คชื่อ:

  • Line messaging
  • Status updates
  • Real-time alerts
Line API Notifications
3

Data Recording

เครืองสแกนทำการบันทึกค่าข้อมูลที่ได้จากนั้นส่งข้อมูลไปเทียบกับฐานข้อมูลใน Google Sheet ที่ได้เตรียมไว้เพื่อดูว่าบัตรที่สแกนมีในฐานข้อมูลหรือไม่ ถ้าไม่มีให้แสดง "Unknown" พร้อมกับรหัสบัตร:

  • Database lookup
  • Student verification
  • Timestamp recording
Google Sheets Cloud Storage
4

Notification

เครื่องสแกนแสดงผลข้อมูลของนักศึกษาที่สแกนบัตรเข้ามาไว้ใน Google Sheet เพื่อเป็นการบันทึกข้อมูล:

  • Student details
  • Timestamp
  • Attendance status
Google Sheets API Database
